# Project Minify

A mostly vibe-coded plugin to reduce the size of Caido Projects by copying data (e.g. Scopes, Filters, Replay Sessions, M&R Filters) into a new Project.

## Installation guide

- Download the .zip file from https://github.com/saltify7/Project-Minify/releases
- Install to Caido https://docs.caido.io/guides/plugins_installing.html
